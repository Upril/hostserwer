# Video Hosting Service with Spring Boot

This project is a video hosting service built with Spring Boot. 

## Technologies Used

- Java 19
- Spring Boot 3.0.6
- OpenAPI Swagger 3
- Lombok
- JUnit (Tests)

## Features

- Video Sreaming
- Authentication(WIP)
- User Ratings (WIP)
- Watch list (WIP)

## Getting Started

### Prerequisites

- Java 19
- Maven 4.0.0 or later
